# Project Overview: ADyD - TP2 (2026)

This workspace contains materials for **Trabajo Práctico N° 2** of the course **Análisis, Diseño y Documentación de Sistemas (ADyD) - 2026** at Universidad Nacional del Comahue (Uncoma).

The project is academic and non-code based, focusing on software modeling using UML.

## Directory Contents

- **Trabajo Práctico N° 2 (DG y DA) - ADyD 2026.pdf**: The assignment specification. It details 8 different domains (scenarios) from which students must choose one to model.
- **Unidad 4_ Diagrama de Clases.pdf**: Theoretical slides and guidelines for creating Class Diagrams, following UML standards. References include Sommerville (Software Engineering) and Fowler (UML Distilled).

## Assignment Objectives

Students are required to:
1.  **Conceptual Class Diagram**: Identify classes, attributes, main methods, relationships, and cardinality for the chosen domain.
2.  **Activity Diagram**: Design a workflow detailing the general business process and its interaction with the defined classes.

## Domains (Scenarios)

1.  **Conectando Hostels**: Web platform for youth hostels and travelers.
2.  **Distribución de Juegos**: Digital game distribution platform for developers and users.
3.  **Cooperativa de terrenos**: Land cooperative management system for subdivision and sale.
4.  **Líneas Aéreas**: Frequent flyer program for Patagonia Airlines (LAP).
5.  **Canchas de Fútbol**: Soccer field rental and reservation system.
6.  **Donaciones Solidarias**: Platform for donating and distributing objects to people in need.
7.  **Alquiler de vehículos**: Peer-to-peer vehicle rental ecosystem (Vehitinder).
8.  **Consultora de RRHH**: Human Resources consultancy system for candidate search and selection.

## Usage

This directory serves as a repository for the assignment requirements and reference materials. Future work in this workspace will likely involve creating the requested diagrams or documenting the analysis for one of the domains.
